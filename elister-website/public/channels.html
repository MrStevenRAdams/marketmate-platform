Integrations

Every channel that matters

e-Lister provides deep, native integrations — not superficial connections. Each channel integration is built and maintained by our engineering team, not a third-party connector.

⏰ Amazon integration is scheduled for release on 30 May 2026. All other channels listed below are live and available now.

Marketplaces — Live

Live marketplace integrations

eBay
Marketplace
Live

One of the world's largest online marketplaces, eBay is a cornerstone channel for most multichannel sellers. e-Lister's eBay integration goes beyond basic listing — we support full inventory management, order sync, category-specific attribute schemas and eBay's business policies framework.

  • OAuth authentication — secure, no API key sharing
  • Full listing create, update and delete via eBay Inventory API
  • AI-powered eBay-specific title, description and category generation
  • eBay category schema sync — attributes updated automatically as eBay changes its taxonomy
  • Business policies integration (payment, returns, shipping)
  • Order import and status updates (tracking number feed-back)
  • Buyer messaging — receive and reply to eBay messages in e-Lister
  • Listing analytics — views, watchers and sale conversion data
Etsy
Marketplace
Live

The global marketplace for handmade, vintage and craft supplies. Etsy has its own taxonomy, tagging conventions and listing requirements that differ significantly from other platforms. e-Lister's Etsy integration handles these nuances natively.

  • PKCE OAuth flow — the most secure Etsy authentication method
  • Full taxonomy access including node-level property definitions
  • Listing create, update, delete and status management
  • Etsy-specific AI listing generation (tags, personalisation options, materials)
  • Shipping profile integration
  • Image upload and management via the Etsy API
  • Order import and tracking number submission
  • Shop-level data and listing metrics
OnBuy
Marketplace
Live

The UK's fastest-growing independent marketplace. OnBuy operates on a competitive, catalogue-based model where sellers compete on price and availability for shared product listings. e-Lister makes it easy to manage your OnBuy offers alongside your other channels.

  • API key authentication with connection test
  • Category browse and product condition management
  • Listing draft preparation and submission from your PIM
  • Listing update and deletion
  • Paginated listing retrieval and status monitoring
  • Order import and fulfilment status sync
Kaufland
Marketplace
Live

One of Germany's leading online marketplaces, Kaufland.de (formerly Real.de) provides access to millions of German shoppers. Essential for sellers targeting the European market. e-Lister's Kaufland integration handles the German marketplace's unit-based listing model.

  • API key authentication with connection testing
  • Full Kaufland category tree navigation
  • Unit (product) creation, update, deletion
  • Listing draft preparation from your product catalogue
  • Seller unit management and status monitoring
  • Order import and fulfilment status reporting

Marketplace

If you sell electronics, Back Market is a high-value channel and e-Lister makes it simple to manage alongside your others.

  • Full Back Market API integration

Mirakl Marketplace Network

Upcoming integration

Amazon
Marketplace
Launching 30 May 2026

Amazon is the world's largest online marketplace. e-Lister's Amazon integration is currently in development and will launch on 30 May 2026. It will be built on the Amazon Selling Partner API (SP-API) and will provide a comprehensive suite of features for Amazon sellers.

  • Selling Partner API (SP-API) integration
  • Product listing creation and management across Amazon marketplaces
  • Amazon category schema and attribute mapping
  • Order import and status management
  • FBA (Fulfilment by Amazon) inbound shipment management
  • Real-time inventory sync including FBA stock levels
  • Amazon buyer messaging via SP-API Messaging API
  • Listing analytics and performance data
