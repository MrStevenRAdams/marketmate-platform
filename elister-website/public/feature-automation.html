Automation Features

If this, then that — for your entire operation

🎯 Event-Based Triggers

Trigger rules on any platform event: new order received, order dispatched, stock falls below threshold, listing published, buyer message received, carrier exception raised, purchase order created, and more.

🔧 Condition Builder

Combine multiple conditions with AND/OR logic. Test against order value, channel, SKU, stock level, customer country, carrier service, product category, tag presence, and dozens of other fields.

⚡ Multi-Action Rules

One rule can execute multiple actions simultaneously: assign a carrier, apply a tag, send a message, raise a PO, and update a field — all from a single trigger event. Cross-module rules are a MarketMate exclusive.

📜 Automation Logs

Every rule execution is logged with full context: which rule fired, why it matched, what actions were taken, and the result. View, filter, and retry failed automation runs from the Automation Logs page.

🛤️ Visual Workflow Builder Q3 2026

Our drag-and-drop workflow canvas lets you build complex branching decision trees visually. See the full logic flow of your automation before activating it — no SQL, no code, no developer tickets.

📨 Buyer Messaging Automation Q3 2026

Create email and message templates with dynamic field insertion (order number, item name, tracking number, expected delivery). Trigger template sends automatically on order status changes or SLA events.

Example Automations

Rules our customers run every day

📦 Inventory Rule

When stock falls below 10 units, raise a draft purchase order to the preferred supplier with the reorder quantity calculated from the 30-day sales average.

🚚 Carrier Rule

Assign DPD Next Day to all Amazon Prime orders over £30 received before 2pm. Assign Royal Mail Tracked 48 to all non-Prime orders.

💬 Messaging Rule

When an order is marked dispatched on eBay, automatically send a personalised dispatch confirmation message with the tracking number using the approved eBay messaging template.

🚨 Alert Rule

When an order's dispatch-by time is less than 2 hours away and it's still Awaiting, tag it as URGENT, move it to the Priority folder, and notify the warehouse manager by email.
